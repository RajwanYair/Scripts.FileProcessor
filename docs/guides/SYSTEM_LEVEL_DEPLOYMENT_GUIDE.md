## 📋 Production Deployment Instructions

### Windows Deployment
```cmd
# System requirements check
python --version  # Should be 3.9+

# Install core dependencies (system-level)
pip install -r requirements.txt

# Verify installation
python enhanced_dependency_manager.py

# Launch application
python enhanced_gui_v5.py
```

### Linux/WSL Deployment
```bash
# Install via system package manager (preferred)
sudo apt install python3-pip python3-aiofiles python3-psutil python3-pil python3-yaml

# Or install via pip (system-level)
pip install -r requirements.txt

# Verify installation
python3 enhanced_dependency_manager.py

# Launch application
python3 file_processing_suite_main.py --help
```

### macOS Deployment
```bash
# Install dependencies (system-level)
pip install -r requirements.txt

# Verify installation
python enhanced_dependency_manager.py

# Launch application
python enhanced_gui_v5.py
```
